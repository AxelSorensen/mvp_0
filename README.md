# AIBD - MVP

## Setup

Install dependencies

```bash
# npm
npm install
```

Create .env file with OpenAI API key in following format:
```bash
OPENAI_API_KEY="..."
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
